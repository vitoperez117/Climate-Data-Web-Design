City Weather Data

Explanation of Project

We gathered weather data from over 500 cities around the world to determine the effect of latitude on meteorological observations.

Using OpenWeatherMap API, we gathered weather data from each city. We extracted readings on temperature, cloudinees, humidity, and wind speed, then compiled them into a Data Frame. Using Matplotlib, we created scatterplots to ascertain possible correlations between latitude and weather data gathered.

Included in this site are visualizations, explanations with analysis per visualization, as well as the source data set.
